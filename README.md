## 🇬🇧 English Version

> Smarter, real-time inventory control.

MyLogist is an inventory management system (SaaS) designed to optimize stock control, centralize product information, and provide accurate metrics for logistics decision-making. Built with a robust cloud architecture to ensure high availability and performance.

### ✨ Key Features

*   **Real-Time Data:** Instant stock updates to prevent inventory shortages.
*   **Centralized Management:** Control suppliers, inbound, and outbound merchandise from a single dashboard.
*   **Advanced Search & Filtering:** Find specific products and movements in seconds.
*   **Smart Alerts:** Automatic notifications for low stock or stagnant merchandise.
*   **Impact Metrics:** Clear reports to analyze logistical and financial performance.

### 🛠️ Tech Stack

**Frontend (Landing Page & App):**
*   Next.js (App Router)
*   React
*   Tailwind CSS

**Infrastructure & Backend:**
*   Amazon Web Services (AWS) - High availability and deployment.
*   Docker - Environment containerization.
*   Vercel - Optimized frontend deployment.

### 🚀 Local Installation & Deployment

Follow these steps to set up the local development environment:

1. Clone the repository:
    `git clone https://github.com/YOUR-USERNAME/mylogist.git`
    `cd mylogist`

2. Install dependencies (Frontend):
    `npm install`

3. Configure environment variables:
    Create a `.env.local` file in the root directory and add the necessary variables (see `.env.example`).

4. Start the development server:
    `npm run dev`

The application will be available at `http://localhost:3000`.

## 🇪🇸 Versión en Español

> El control de tu inventario, más inteligente y en tiempo real.

MyLogist es un sistema de gestión de inventarios (SaaS) diseñado para optimizar el control de stock, centralizar la información de productos y proporcionar métricas precisas para la toma de decisiones logísticas. Construido con una arquitectura robusta en la nube para garantizar alta disponibilidad y rendimiento.

### ✨ Características Principales

*   **Datos en Tiempo Real:** Actualización instantánea del stock para evitar quiebres de inventario.
*   **Gestión Centralizada:** Control de proveedores, entradas y salidas de mercadería desde un único panel.
*   **Búsqueda y Filtrado Avanzado:** Encuentra productos y movimientos específicos en segundos.
*   **Alertas Inteligentes:** Notificaciones automáticas por stock bajo o mercadería estancada.
*   **Métricas de Impacto:** Reportes claros para analizar el rendimiento logístico y financiero.

### 🛠️ Stack Tecnológico

**Frontend (Landing Page & App):**
*   Next.js (App Router)
*   React
